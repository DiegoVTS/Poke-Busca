# PokéBusca 
> Projeto integrador desenvolvido para praticar JavaScript, consumo de API, manipulação de DOM e async/await.

Buscador de Pokémon desenvolvido com HTML, CSS e JavaScript puro.

## 📋 Sobre o Projeto

Projeto integrador desenvolvido para praticar JavaScript, consumo de API, manipulação de DOM e async/await. O usuário digita o nome de um Pokémon e o site retorna as informações em tempo real usando a PokeAPI.

## ✨ Funcionalidades

- Busca de Pokémon por nome
- Exibe Nome, HP, tipo e Imagem do Pokémon
- Lista dos últimos 5 Pokémons buscados
- Mensagem de erro para Pokémon não encontrado

## 🛠️ Tecnologias

- HTML5 Semântico
- CSS3 — Flexbox, Grid, animações e transições
- JavaScript — DOM, Eventos, Fetch, Async/Await
- [PokeAPI](https://pokeapi.co/)
